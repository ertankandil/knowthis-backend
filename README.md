# KnowThis AI Detection Backend

AI voice detection API for KnowThis iOS app.

## Features
- Audio file upload and analysis
- Real-time AI voice detection using librosa
- MFCC, spectral, and pitch analysis

## Deploy to Railway
1. Push to GitHub
2. Connect repo to Railway
3. Deploy automatically

## API Endpoints
- `GET /` - Health check
- `POST /analyze` - Analyze audio file
